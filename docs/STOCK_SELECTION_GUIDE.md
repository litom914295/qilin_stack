**适用对象**: 一进二短线交易者

## 🎯 一进二交易的本质

### 什么是"一进二"？

**一进二** = **T日涨停（一板）** → **T+1继续涨停（二板）**

```
T日（首板）：
├─ 股票涨停，封板强劲
└─ 预测T+1会继续涨停

T+1日（二板）：
├─ 竞价强势，继续涨停
└─ 实现10-20%收益

T+2日（卖出）：
└─ 高开兑现利润
```

### 为什么要做一进二？

✅ **优势**：
1. **确定性高**：首板涨停，市场已验证该股强势
2. **爆发力强**：二板收益率10-20%，三板更高
3. **周期短**：T日选→T+1买→T+2卖，2天完成一个循环
4. **可复制**：每天都有涨停股，机会多

⚠️ **劣势**：
1. **T+1无法止损**：买入后必须持有到次日
2. **竞价难度大**：超强股竞价就涨停，难买到
3. **市场环境依赖**：弱市成功率大幅下降
4. **情绪波动大**：涨停炸板心态受影响

## 📊 三层过滤选股体系

系统采用**三层漏斗式过滤**，从100+只涨停股筛选出5-10只精品：

```
T日涨停股（100+只）
    ↓
【第一层：基础过滤】淘汰70%
    ↓
剩余30只左右
    ↓
【第二层：质量评分】淘汰50%
    ↓
剩余15只左右
    ↓
【第三层：市场环境】淘汰30%
    ↓
最终5-10只精品
```

## 🔍 第一层：基础过滤（硬指标）

### 1.1 封单强度 > 80

**什么是封单强度？**
- 封单强度 = (封单金额 / 流通市值) × 100
- 衡量买盘力量是否足够强大

**为什么重要？**
- 封单强度高 → 市场认可度高 → T+1继续强势概率大
- 封单强度低 → 容易炸板 → T+1走弱风险高

**筛选标准**：
```
超强: 封单强度 > 100  （一字板级别）
优秀: 封单强度 80-100 （强势板）
及格: 封单强度 50-80  （普通板）
❌ 放弃: 封单强度 < 50  （弱势板）
```

**实战案例**：
```
案例1: 某AI芯片股
- T日封单强度: 120
- T+1开盘: 直接一字涨停
- 结果: 竞价买入困难，但成功率极高

案例2: 某跟风题材股
- T日封单强度: 45
- T+1开盘: 低开-2%
- 结果: T+1继续弱势，亏损5%
```

---

### 1.2 涨停时间 < 10:30

**为什么重要？**
- 早盘涨停 → 资金抢筹积极 → 强势
- 尾盘涨停 → 资金犹豫不决 → 弱势

**筛选标准**：
```
超强: 9:30-9:45  （开盘即涨停，最强）
优秀: 9:45-10:00 （早盘强势）
及格: 10:00-10:30（上午涨停）
⚠️  警惕: 10:30-13:00（午盘涨停，较弱）
❌ 放弃: 13:00-15:00（尾盘涨停，最弱）
```

**实战逻辑**：
- 9:30涨停的股票，说明**集合竞价后资金就疯抢**
- 14:30涨停的股票，说明**资金观望了一整天才动手**
- 谁更强势？显而易见是前者

**特殊情况**：
- 午后炸板后重新封回 → **谨慎**，封单不稳定
- 盘中多次炸板 → **放弃**，分歧太大

---

### 1.3 开板次数 ≤ 2次

**什么是开板？**
- 涨停后价格跌破涨停价，称为"炸板"或"开板"

**为什么重要？**
- 开板次数 = 市场分歧程度
- 开板越多 → 分歧越大 → T+1走弱风险越高

**筛选标准**：
```
最强: 0次开板  （一字板，无分歧）
优秀: 1次开板  （短暂分歧，快速回封）
及格: 2次开板  （有分歧，但最终统一）
❌ 放弃: ≥3次开板（分歧严重，资金犹豫）
```

**开板细节分析**：
```
开板1次的情况：
├─ 开板1分钟内回封 → 优秀（洗盘）
├─ 开板5分钟内回封 → 良好（短暂分歧）
└─ 开板10分钟以上 → 一般（分歧较大）

开板2次的情况：
├─ 都是短暂开板 → 及格
└─ 有长时间开板 → 警惕
```

---

### 1.4 排除ST、*ST、退市股

**为什么排除？**
- ST股票：涨跌停限制仅5%，收益空间小
- 退市风险股：存在退市可能，风险极高
- 业绩亏损股：基本面差，持续性差

**筛选标准**：
```
❌ 绝对排除：
├─ ST、*ST股票
├─ 退市风险警示股
├─ 连续3年亏损股
└─ 被立案调查股

⚠️  谨慎对待：
├─ 业绩预亏股
├─ 商誉减值股
└─ 重大重组失败股
```

## 📈 第二层：质量评分（综合指标）

经过第一层过滤，剩余30只左右。现在用**质量评分系统**打分，筛选出Top 15。

### 质量评分公式（满分100）

```
质量评分 = 封单强度(40分) + 涨停时间(20分) + 板块联动(20分) + 
          换手率(10分) + 资金流向(10分)
```

---

### 2.1 封单强度（40分）

**计算方式**：
```python
封单强度分 = min(封单强度 / 100 * 40, 40)

示例：
- 封单强度100 → 40分（满分）
- 封单强度80  → 32分
- 封单强度50  → 20分
```

---

### 2.2 涨停时间（20分）

**计算方式**：
```python
涨停时间分 = 根据涨停时间分级打分

09:30-09:45  → 20分（满分）
09:45-10:00  → 18分
10:00-10:30  → 15分
10:30-11:30  → 12分
11:30-13:00  → 8分
13:00-14:00  → 5分
14:00-15:00  → 2分
```

**逻辑**：
- 越早涨停，得分越高
- 开盘10分钟内涨停最佳

---

### 2.3 板块联动（20分）

**什么是板块联动？**
- 板块联动 = 同板块涨停股数量
- 衡量该股是否属于市场热点

**为什么重要？**
- 板块联动强 → 题材持续性强 → T+1继续强势
- 板块联动弱 → 孤立个股 → T+1容易分化

**计算方式**：
```python
板块涨停数 = 同板块当日涨停股票数量

板块联动分 = 根据板块涨停数打分：
≥10只 → 20分（强势题材，满分）
7-9只  → 16分（热点题材）
4-6只  → 12分（一般题材）
2-3只  → 8分（弱势题材）
1只    → 4分（孤立个股）
```

**实战应用**：
```
示例1: AI芯片板块
- 当日涨停数: 15只
- 板块联动分: 20分
- 预判: 强势题材，龙头股T+1继续涨停概率高

示例2: 某冷门概念
- 当日涨停数: 1只
- 板块联动分: 4分
- 预判: 孤立个股，T+1走弱风险大
```

---

### 2.4 换手率（10分）

**什么是换手率？**
- 换手率 = 成交量 / 流通股本 × 100%
- 衡量资金活跃度

**为什么重要？**
- 换手充分 → 筹码交换充分 → T+1压力小
- 换手不足 → 筹码没换手 → T+1获利盘抛压大

**计算方式**：
```python
换手率分 = 根据换手率打分：
15-30% → 10分（最佳，活跃且不过热）
10-15% → 8分（良好）
30-50% → 6分（过热，警惕）
5-10%  → 4分（偏低）
<5%    → 2分（太低，不活跃）
>50%   → 2分（过热，风险大）
```

**最佳区间**：
- **15-30%**：资金活跃，换手充分，又不过度投机

**警惕信号**：
- **<5%**：筹码锁定太好，T+1可能缺乏承接
- **>50%**：过度投机，T+1容易分化

---

### 2.5 资金流向（10分）

**什么是资金流向？**
- 主力净流入 = 大单买入额 - 大单卖出额
- 衡量主力资金态度

**计算方式**：
```python
资金流向分 = 根据主力净流入额打分：
>1亿    → 10分（主力大幅流入）
5000万-1亿 → 8分（主力流入）
1000万-5000万 → 6分（一般）
0-1000万 → 4分（弱势）
<0（净流出）→ 2分（主力出逃）
```

**警惕信号**：
- **主力净流出** + **涨停**：可能是诱多，谨慎

---

**案例1: 某AI龙头股（优质）**
```
封单强度: 120  → 40分
涨停时间: 9:35  → 20分
板块联动: 15只 → 20分
换手率: 22%    → 10分

━━━━━━━━━━━━━━━━━━━━

```

```
━━━━━━━━━━━━━━━━━━━━
总分: 38分 ❌
评价: 弱势股，放弃
```

**筛选标准**：
```
质量评分 ≥ 85分 → 优质候选
质量评分 70-85分 → 一般候选
质量评分 < 70分 → 放弃
```

## 🌐 第三层：市场环境（宏观过滤）

即使个股质量评分高，也要看市场环境是否支持。

### 3.1 市场涨停数

**筛选标准**：
```
活跃市场: >100只 → 环境优秀，积极参与
正常市场: 50-100只 → 环境正常，正常参与
弱势市场: 30-50只 → 环境一般，谨慎参与
❌ 极弱市场: <30只 → 环境恶劣，观望为主
```

**逻辑**：
- 涨停数多 → 市场情绪好 → 赚钱效应强 → 成功率高
- 涨停数少 → 市场情绪差 → 分化严重 → 成功率低

**实战建议**：
```
涨停数<30只时：
├─ 即使有90分的股票，也建议观望
├─ 或者仅买1-2只龙头，仓位降低到3-5%
└─ 风控第一，活着最重要
```

---

### 3.2 大盘走势

**筛选标准**：
```
强势大盘: 大盘涨>1% → 助涨，积极参与
正常大盘: 大盘涨跌±1% → 中性，正常参与
弱势大盘: 大盘跌1-2% → 拖累，谨慎参与
❌ 暴跌大盘: 大盘跌>2% → 系统性风险，观望
```

**逻辑**：
- 大盘涨 → 情绪好 → 个股涨停持续性强
- 大盘跌 → 情绪差 → 个股容易分化炸板

**特殊情况**：
- **大盘暴跌但某板块逆势大涨** → 强势题材，可参与龙头

---

### 3.3 板块轮动

**判断标准**：
```
明确主线: 某板块连续3天领涨 → 题材持续性强
板块轮动: 每天不同板块轮流涨 → 持续性差
板块分化: 板块内部分化严重 → 龙头跟风差距大
❌ 无主线: 市场散乱，无明确热点 → 观望
```

**如何判断主线明确？**
1. 某板块连续多日涨停股最多
2. 龙头股连板加速（三板、四板、五板）
3. 跟风股也有较好表现

**实战建议**：
```
主线明确时：
├─ 优先买龙头股（连板股）
├─ 其次买核心跟风股（首板强势股）
└─ 避开冷门题材

无主线时：
├─ 降低仓位至30-40%
├─ 只买质量评分90分以上的
└─ 随时准备空仓观望
```

---

### 3.4 市场情绪指数

**情绪指数计算**（系统自动）：
```python
情绪指数 = (涨停数 / (涨停数 + 跌停数)) × 100

分级：
90-100分 → 情绪极佳（疯牛）
70-90分  → 情绪良好（牛市）
50-70分  → 情绪正常（震荡）
30-50分  → 情绪偏弱（熊市）
0-30分   → 情绪极差（暴跌）
```

**实战应用**：
```
情绪极佳（90+）：
├─ 加大仓位至60-80%
├─ 积极参与龙头+跟风
└─ 注意追高风险

情绪正常（50-70）：
├─ 正常仓位40-50%
├─ 精选龙头
└─ 标准策略

情绪极差（<30）：
├─ 空仓观望
├─ 不要抄底
└─ 等待市场企稳
```

## 🎯 竞价进阶优化（系统独有）

经过三层过滤后，还需要在T日盘后用**竞价进阶模块**做最后优化。

### 竞价进阶做什么？

1. **数据质量审计**
   - 检查候选股数据完整性
   - 排查缺失值、异常值
   - 确保特征覆盖率>95%

2. **核心特征筛选**
   - 从100+特征中筛选50个核心特征
   - 去除噪声特征
   - 提高模型预测稳定性

3. **因子衰减监控**
   - 监控15+润停核心因子表现
   - 自动调整因子权重
   - 识别衰减因子并降权

4. **Walk-Forward回测**
   - 滚动训练模型（180天训练、60天测试）
   - 评估模型稳定性（AUC>0.70）
   - 确保模型泛化能力

5. **市场情绪分析**
   - 综合市场情绪指数
   - 板块轮动强度
   - 资金流向趋势

### 竞价进阶的价值

**没有竞价进阶**：
- 只能靠人工筛选
- 主观判断占比大
- 成功率60-65%

**有竞价进阶**：
- 系统化量化筛选
- 模型预测辅助决策
- 成功率提升至70-75%

## 💰 T+1竞价强度分级（最终决策）

T+1日早上9:15-9:25，实时监控竞价数据，按**竞价强度**分级决策。

### 竞价强度计算（满分100）

```python
竞价强度 = 竞价涨幅(40分) + 买卖单比(30分) + 
          大单占比(20分) + 价格稳定性(10分)
```

---

### 竞价强度分级

#### Level 1: 超强（≥85分）

**特征**：
- 竞价涨幅：>5%（接近或达到涨停）
- 买卖单比：>2:1（买盘占比67%+）
- 大单占比：>60%
- 价格稳定：波动<1%

**决策**：
```
策略: 竞价结束立即买入（9:25前下单）
价格: 涨停价（不怕买贵）
仓位: 单股8-10%（重仓）
理由: 确定性极高，大概率一字板或T字板
风险: 中等（虽强势但仍有T+1风险）
```

**实战案例**：
```
某AI龙头股（2024-03-15）：
- 竞价涨幅: +9.8%（涨停）
- 买卖单比: 10:1
- 竞价强度: 95分
- 操作: 9:24:50下单，涨停价买入
- 结果: 开盘一字涨停，T+2高开6%卖出，盈利16%
```

---

#### Level 2: 强势（70-85分）

**特征**：
- 竞价涨幅：3-5%
- 买卖单比：1.5-2:1（买盘占比60-67%）
- 大单占比：40-60%
- 价格稳定：波动1-2%

**决策**：
```
策略: 开盘观察30秒-1分钟后买入
价格: 竞价价-0.5%（等回踩）
仓位: 单股5-8%（中等仓位）
理由: 强势但不极端，开盘可能小幅回踩
风险: 中等
```

**实战案例**：
```
某新能源跟风股（2024-03-18）：
- 竞价涨幅: +4.2%
- 买卖单比: 1.8:1
- 竞价强度: 78分
- 操作: 开盘观察，9:31回踩至+3.5%买入
- 结果: 盘中冲至+7%，T+2平开卖出，盈利10%
```

---

#### Level 3: 中等（55-70分）

**特征**：
- 竞价涨幅：1-3%
- 买卖单比：1-1.5:1（买盘占比50-60%）
- 大单占比：20-40%
- 价格稳定：波动2-3%

**决策**：
```
策略: 等待回踩2-3%后买入（可能买不到）
价格: 竞价价-3%
仓位: 单股3-5%（小仓位）
理由: 走势不明朗，需要大回踩才有性价比
风险: 高（可能走弱，也可能直接涨停买不到）
```

**实战案例**：
```
某题材股（2024-03-20）：
- 竞价涨幅: +2.1%
- 买卖单比: 1.3:1
- 竞价强度: 62分
- 操作: 等待回踩，但直接高开高走，放弃
- 结果: 未买入，盘中涨至+5%
- 总结: 错过但无损失，符合风控原则
```

---

#### Level 4: 弱势（<55分）

**特征**：
- 竞价涨幅：<1%或负值
- 买卖单比：<1:1（卖盘更多）
- 大单占比：<20%
- 价格稳定：波动>3%

**决策**：

## 🚫 选股的10大禁忌

### 1. 不要追涨停板数（数量≠质量）
❌ **错误**：今天涨停100只，随便选10只  
✅ **正确**：严格按三层过滤，宁缺毋滥

### 2. 不要忽略竞价表现
❌ **错误**：T日选得好，T+1竞价弱势也买  
✅ **正确**：竞价是最后防线，弱势果断放弃

### 3. 不要逆市操作
❌ **错误**：大盘暴跌，涨停数<30只，仍满仓  
✅ **正确**：市场环境恶劣时，空仓观望

### 4. 不要追高尾盘涨停
❌ **错误**：14:30涨停的股票照买  
✅ **正确**：尾盘涨停持续性差,放弃

### 5. 不要买炸板3次以上的股
❌ **错误**：最终封板就行  
✅ **正确**：炸板3次说明分歧太大，T+1容易走弱

### 6. 不要买孤立个股
❌ **错误**：板块只涨停1只，也买  
✅ **正确**：板块联动差，持续性存疑

### 7. 不要买ST、退市股
❌ **错误**：ST股涨停也是涨停  
✅ **正确**：ST股涨停仅5%，风险收益比不划算

### 8. 不要买主力净流出的涨停股
❌ **错误**：涨停就是强势  
✅ **正确**：涨停+主力净流出可能是诱多

### 9. 不要忽略大盘走势
❌ **错误**：大盘跌3%，个股照买  
✅ **正确**：大盘暴跌拖累个股，成功率大降

### 10. 不要情绪化选股
❌ **错误**：看到热点就追，看到亏损就慌  
✅ **正确**：严格按系统筛选，保持纪律性

## 💡 高手进阶：看盘技巧

### 盘口细节观察

#### 1. 封单结构
```
优质封板：
├─ 封单笔数多（>1000笔）
├─ 大单占比高（>60%）
├─ 封单持续增加
└─ 买盘队列长（>10万手）

劣质封板：
├─ 封单笔数少（<500笔）
├─ 散户为主（大单<30%）
├─ 封单时增时减
└─ 买盘队列短（<5万手）
```

#### 2. 分时形态
```
强势形态：
├─ 一字板：最强，无分歧
├─ T字板：强势，短暂分歧后回封
├─ 火箭板：直线拉升封板
└─ 缩量板：筹码锁定好

弱势形态：
├─ 尾盘板：全天犹豫，尾盘才封
├─ 放量板：换手太大，抛压重
├─ 炸板多次：分歧严重
└─ 开板时间长：封单不稳
```

#### 3. 龙虎榜
```
看涨信号：
├─ 机构席位买入
├─ 游资一致性买入
├─ 卖方无机构席位
└─ 买入金额远大于卖出

看跌信号：
├─ 机构席位卖出
├─ 游资席位对倒
├─ 卖方有大笔抛售
└─ 卖出金额大于买入
```

### 题材判断

#### 题材生命周期
```
萌芽期（1-2天）：
├─ 首次出现，涨停数少（5-10只）
├─ 市场认知度低
├─ 策略：观望，等龙头确认
└─ 风险：题材可能夭折

爆发期（3-5天）：
├─ 涨停数激增（20-50只）
├─ 龙头连板加速（三板、四板）
├─ 策略：积极参与龙头+核心跟风
└─ 风险：追高风险上升

高潮期（6-8天）：
├─ 全市场关注，涨停数峰值
├─ 龙头连板（五板、六板、七板）
├─ 策略：谨慎参与，随时准备撤退
└─ 风险：高位分化风险极大

衰退期（9天+）：
├─ 涨停数骤减，分化严重
├─ 龙头炸板或换手
├─ 策略：不再新进，已持有的考虑撤退
└─ 风险：成为接盘侠
```

#### 题材持续性判断
```
强持续性：
├─ 政策持续催化（国家级政策）
├─ 业绩持续兑现（订单、利润增长）
├─ 市场情绪持续高涨
└─ 示例：AI芯片、新能源（2023-2024）

弱持续性：
├─ 事件性催化（突发新闻）
├─ 概念性炒作（无业绩支撑）
├─ 市场情绪快速衰退
└─ 示例：某些短期热点、突发概念
```
