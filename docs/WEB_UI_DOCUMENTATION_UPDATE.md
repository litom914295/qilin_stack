## 关键改进点

### 1. 统一的文档引用格式
所有页面都使用统一的格式引用改进文档：
```markdown
**🆕 系统改进文档** (最新):
- 🦄 **麒麟改进实施报告**: `docs/QILIN_EVOLUTION_IMPLEMENTATION.md`
```

### 2. 分层次的指南设计
- **新手用户**: 查看"使用步骤"和"工作流程"
- **进阶用户**: 查看"核心功能"和"相关文档"
- **开发者**: 查看代码文件路径

### 3. 可折叠的设计
所有使用指南都使用`st.expander`实现可折叠，默认折叠状态，不影响主界面简洁性。

### 4. 闭环说明
每个页面都包含与其他功能模块的联动说明，形成完整的工作闭环：
- AI进化系统 → 循环进化训练 → 写实回测 → 智能预测 → 交易执行

## 快速访问路径

用户在任何功能页面都可以通过以下方式快速找到完整文档：

1. **Web界面内**:
   - 点击页面顶部的"📖 系统使用指南"
   - 查看"🆕 系统改进文档"区块

2. **侧边栏**:
   - "📚 文档与指南" → 选择预览文档

3. **项目根目录**:
   - 查看 `HOW_TO_USE.md` 获取总览
