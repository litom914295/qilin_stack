# 🚀 Phase 1 快速开始指南

## 📦 新功能概览

Phase 1 为**一进二涨停监控系统**带来了全新的统一界面：

- ✅ **统一入口** - 不再需要在多个菜单间跳转
- ✅ **智能阶段识别** - 自动判断当前处于哪个交易阶段
- ✅ **核心指标可视化** - 一眼看清候选/监控/持仓/盈亏
- ✅ **业务流程导向** - T日选股 → T+1竞价 → T+2卖出，流程清晰

---

## 🎯 如何使用

### 1. 启动应用

```bash
cd G:/test/qilin_stack
streamlit run web/unified_dashboard.py
```

浏览器会自动打开 `http://localhost:8501`

### 2. 进入新界面

在主页面顶部，点击 **🎯 一进二涨停监控** 标签页（第一个标签）

### 3. 查看当前状态

**顶部区域会显示：**
- **当前交易阶段** - 如：T+1盘中交易，距离收盘 3小时44分钟
- **智能建议** - 根据当前阶段给出操作建议
- **核心指标** - 候选池、监控中、持仓、盈亏

### 4. 根据阶段选择Tab

界面下方有4个tab，根据你当前的需求选择：

#### 📊 T日选股
- **使用时机**: 收盘后（15:00后）
- **功能**: 筛选今日涨停股，构建明日监控池
- **亮点**: 三层筛选漏斗（基础过滤→质量评分→AI选股）

#### 🔥 T+1竞价监控
- **使用时机**: 集合竞价期间（9:15-9:30）
- **功能**: 实时监控候选池竞价表现
- **亮点**: AI决策结果，竞价强度分布

#### 💰 T+2卖出决策
- **使用时机**: 持仓日的开盘前后
- **功能**: 持仓管理，止盈止损建议
- **亮点**: 卖出策略建议（开发中）

#### 📈 统计分析
- **使用时机**: 任何时候
- **功能**: 查看历史回测、参数优化、涨停原因分析

## 💡 使用技巧

### 阶段自动识别

系统会根据当前时间自动判断交易阶段：

| 时间段 | 阶段 | 颜色 |
|--------|------|------|
| 09:15前 | T日选股 | 🟦 蓝色 |
| 09:15-09:30 | T+1竞价监控 | 🟢 绿色 |
| 09:30-15:00 | T+1盘中交易 | 🟡 黄色 |
| 15:00后 | T+2卖出决策 | 🟣 紫色 |

### 智能提示

系统会根据数据给出个性化提示，例如：
- 候选池数量过多 → 建议进一步筛选
- 竞价强势股数 > 3 → 建议优先买入
- 持仓盈利 → 建议适时止盈

## ❓ 常见问题

### Q: 为什么显示"未找到数据"？

**A**: Phase 1 主要是界面重构，需要实际的涨停数据。你可以：
1. 运行 `python app/daily_workflow.py` 生成今日数据
2. 或者等待下一个交易日自动生成

### Q: 旧版界面还能用吗？

**A**: 可以！旧版在 `🔖 竞价决策(旧)` 标签页，保留用于对比。

### Q: 新版有哪些不同？

**A**: 
- ✅ 入口更统一（1个 vs 3个）
- ✅ 流程更清晰（业务导向 vs 技术导向）
- ✅ 阶段更智能（自动识别 vs 手动判断）
- ✅ 指标更直观（4个核心指标一目了然）
