# 📋 一进二操盘SOP工作流

> **标准操作流程** - 从盘前准备到盘后复盘的完整日常工作指南

**版本**: v1.0  
**更新日期**: 2025-10-30  
**适用对象**: 一进二短线交易者

---

## 📅 完整三日操作周期

```
T日（盘后）→ 筛选候选 + 优化分析
    ↓
T+1日（盘中）→ 竞价监控 + 买入决策
    ↓
T+2日（盘中）→ 卖出执行 + 盘后复盘
```

---

### 📍 步骤4：人工复核与调整（15:30-16:30）

**重点检查项**：

#### 1. 个股基本面筛查（必做）
对每只候选股进行快速检查：

**排除标准**：
- ❌ ST、*ST股票
- ❌ 业绩大幅亏损
- ❌ 重大利空消息
- ❌ 停牌风险
- ❌ 退市风险警告

**加分标准**：
- ✅ 热门题材（AI、新能源、芯片等）
- ✅ 龙头地位
- ✅ 资金持续流入
- ✅ 机构关注度高

#### 2. 技术面分析（选做）
- 封单强度：>5.0为强势
- 换手率：15-30%为活跃
- 涨停时间：早盘涨停优于尾盘
- 分时形态：一字板>T字板>普通涨停

#### 3. 市场环境评估（必做）

**整体市场**：
- 大盘走势：强势市场成功率高
- 两市涨停数：>100只为活跃
- 市场情绪：竞价进阶中查看"市场情绪评分"

**板块轮动**：
- 是否有明确的热点板块
- 题材持续性如何
- 龙头股是否涨停

#### 4. 最终确定监控清单

从30只候选中精选**10-15只**作为核心监控：

**优先级排序**：
1. 预测分数 >0.80 + 封单强度 >5.0
2. 热门题材 + 龙头地位
3. 资金流入 + 机构关注
4. 技术形态良好

**记录方式**：
- 导出候选列表CSV
- 或手动记录到Excel
- 备注每只股票的关注理由

### 📍 步骤5：准备次日策略（16:30-17:00）

#### 1. 制定买入计划

**总资金分配**：
```
总资金: 100万（示例）
├─ 单股最大仓位: 10万（10%）
├─ 总仓位上限: 80万（80%）
└─ 预留资金: 20万（20%，应对机会或风险）
```

**分层买入策略**：
```
Layer 1（竞价买入）：
- 条件：竞价超强（买盘占比>80%）
- 仓位：单股8-10万
- 目标：3-5只

Layer 2（开盘后买入）：
- 条件：竞价强势（买盘占比60-80%）
- 仓位：单股5-8万
- 目标：3-5只

Layer 3（回调买入）：
- 条件：开盘后回调不破9:30价位
- 仓位：单股3-5万
- 目标：2-3只
```

#### 2. 设置预警提醒

**竞价监控时间**：
- 09:15-09:20：观察竞价数据
- 09:20-09:24：最终决策窗口
- 09:24-09:25：下单时间

**设置手机闹钟**：
- 09:10 起床准备
- 09:14 打开系统
- 09:20 开始决策

#### 3. 风险预案

**市场风险**：
- 大盘大幅低开（>-1%）：减少买入或观望
- 两市涨停数<50只：谨慎参与
- 板块分化严重：只买龙头

**个股风险**：
- 竞价弱势（买盘<50%）：放弃
- 大幅低开（>-3%）：放弃
- 异常波动：暂停观察

## 🕘 T+1日：盘中监控（09:35-15:00）

### 持仓监控（全天）

**⚠️ 重要提醒**：
- **A股是T+1交易制度**：T+1买入的股票，当天**不能卖出**
- T+1日只能观察持仓走势，评估次日（T+2）的卖出策略
- 无论盘中涨跌如何，都**无法操作**，只能等到T+2日

**操作路径**：
```
🎯 竞价决策 → 💰 买入决策
```

**检查频率**：
- 09:35-10:30：每10分钟检查一次
- 10:30-14:30：每30分钟检查一次
- 14:30-15:00：每10分钟检查一次

**监控目的**：
- 📊 观察股票走势，评估质量
- 📝 记录表现数据，为T+2决策做准备
- 🎯 识别强势股和弱势股
- ⚠️ 发现异常情况（重大利空、突发事件）

**监控指标**：

1. **价格位置**：
   - 涨停：优秀，T+2可考虑部分兑现
   - 7-9%：良好，T+2根据开盘决定
   - 3-7%：一般，T+2倾向全卖
   - 0-3%：弱势，T+2开盘即卖
   - <0%：亏损，T+2必须止损

2. **封单情况**（如涨停）：
   - 封单额>5000万：强势，连板概率高
   - 封单额2000-5000万：中等
   - 封单额<2000万：弱势，容易炸板
   - 炸板次数：炸板越多越弱势

3. **成交量**：
   - 持续放量：资金追捧，看好
   - 量价配合：健康走势
   - 放量滞涨：警惕次日走弱

4. **分时形态**：
   - 一字涨停：最强
   - T字涨停：强势
   - 冲高回落：较弱
   - 震荡下行：最弱

**记录表格**（为T+2决策准备）：

```
股票代码  买入价  收盘价  涨跌幅  封单强度  成交量  分时形态  T+2策略预判
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
000001   10.00   11.00   +10%   超强      放量    一字板    高开兑现50%
000002   15.00   16.20   +8%    强势      正常    T字板     高开卖60%
000003   20.00   20.60   +3%    一般      缩量    震荡      开盘全卖
000004   8.00    7.60    -5%    弱势      放量    下跌      止损全卖
```

**特殊情况应对**：

虽然T+1日不能卖出，但如果出现以下情况，需要做好T+2紧急处理准备：

```
📛 重大利空（必须关注）：
├─ 公司公告重大利空（业绩爆雷、违规处罚等）
├─ 行业政策打压（监管、限制等）
├─ 市场系统性风险（指数暴跌、金融危机等）
└─ 对策：T+2竞价或开盘立即卖出，不要犹豫

⚠️ 大盘暴跌（需要警惕）：
├─ 大盘跌>3%：次日大概率低开
├─ 两市跌停潮：情绪极差
└─ 对策：T+2降低预期，优先兑现利润
```

**心态管理**：

- ✅ 涨停后保持冷静，不要过度兴奋
- ✅ 亏损后保持理性，不要恐慌
- ✅ 记住：无论涨跌，今天都无法操作
- ✅ 专注：观察和记录，为明天做准备
- ❌ 不要：盯盘焦虑、频繁刷新、情绪化

- 如果T+2继续一字涨停：无法卖出，继续持有
- 如果T+2高开后迅速回落：开盘后立即挂单

   - 未成交：立即挂单或市价卖出

2. **补充操作**：
   - 未卖出的50%仓位：设置止盈止损
   - 继续观察走势：如走弱及时卖出

## ⚠️ 风险管理原则

### 1. 仓位管理

```
总资金: 100%
├─ 单股上限: 10%（最多10只）
├─ 总仓位上限: 80%
├─ 预留资金: 20%（应对机会或风险）
└─ 单日最大亏损: 2%（触发后停止交易）
```

### 2. 止损纪律

**硬止损**（必须执行）：
- 个股亏损>7%：立即止损
- 大盘暴跌>3%：减仓50%
- 系统性风险：清仓观望

**软止损**（灵活执行）：
- 午后炸板不回封：卖出50%
- 尾盘跳水>3%：卖出50-100%
- 题材退潮：逐步减仓

### 3. 市场环境判断

**不参与的情况**：
- ❌ 大盘连续暴跌
- ❌ 两市涨停数<30只
- ❌ 无明确主线题材
- ❌ 自己状态不佳

**减仓操作的情况**：
- ⚠️ 大盘单日跌>2%
- ⚠️ 个股竞价弱势
- ⚠️ 题材分化严重

**满仓操作的情况**：
- ✅ 大盘强势突破
- ✅ 涨停数>100只
- ✅ 主线题材明确
- ✅ 龙头连板加速

## 💡 实战技巧

### 1. 选股技巧

- 优先选择市场主线题材
- 关注龙头股和跟风股
- 避开ST、*ST和问题股
- 注重封单强度和预测分数
- 参考竞价进阶的优化结果

### 2. 买入技巧

- 超强竞价果断买，不要犹豫
- 强势竞价开盘观察后买
- 中等竞价谨慎或放弃
- 弱势竞价坚决放弃
- 遵守仓位管理原则

### 3. 持仓技巧

- 涨停股安心持有到T+2
- 强势股（涨幅5-9%）观察持有
- 弱势股（涨幅<5%）T+2果断卖
- 亏损股立即止损，不要幻想

### 4. 卖出技巧

- 高开兑现是最佳策略
- 不要贪恋连板机会（除非龙头）
- T+2卖出不拖到盘中
- 止损要坚决，止盈要及时

## 📞 紧急情况处理

### 系统故障

**问题**：系统打不开或数据异常
**处理**：
1. 检查网络连接
2. 重启Streamlit服务
3. 查看日志文件排查错误
4. 使用备用方案（Excel手工记录）

### 市场突发事件

**问题**：大盘暴跌、个股重大利空
**处理**：
1. 立即停止买入新股
2. 已持仓股票果断止损
3. 清仓观望，保留现金
4. 等待市场稳定后再入场

### 技术故障

**问题**：交易软件无法下单
**处理**：
1. 使用券商APP备用通道
2. 电话委托（拨打券商热线）
3. 记录故障时间和影响
4. 事后投诉和追责
