## 🎯 使用建议

### 在Web界面浏览

1. 启动系统: `streamlit run web/unified_dashboard.py`
2. 点击主界面的 **📚 文档中心** 标签页
3. 使用分类浏览或搜索功能快速定位文档

### 本地阅读

所有文档均为Markdown格式,可使用任何Markdown阅读器:

```bash
# 使用VSCode打开
code docs/

# 或使用Typora等Markdown编辑器
```

### 搜索文档

在文档中心使用全文搜索功能:

- **关键词搜索**: 输入任意关键词,自动搜索所有文档
- **高亮显示**: 搜索结果中关键词会被高亮
- **上下文预览**: 显示匹配行的上下文内容
